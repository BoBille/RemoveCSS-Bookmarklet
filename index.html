RemoveCSS Bookmarklet

Very simple bookmarklet to remove all linked css styling form a site. I'm not sure if this bookmarklet is useful for anything, but it does make it easy to se how a site looks without css styling.

What it does not do is remove inline styling. If you try it on this site, you will see that the "Fork me on GitHub" banner is positioned with inline css and is not affected like the rest.

Install

Just drag this link to you bookmarks bar: RemoveCSS

The javascript

(function(){
  for( i = document.styleSheets.length-1; i>=0; i--){
    document.styleSheets[i].disabled = true;
  }
})();
